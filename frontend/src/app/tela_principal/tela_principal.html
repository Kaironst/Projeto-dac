BANTADS

Sistema de Internet Banking para aqueles associados com o curso de TADS da UFPR.

Sobre o Sistema

Este projeto foi desenvolvido como parte do curso de Análise e Desenvolvimento de Sistemas da UFPR. O objetivo é simular um ambiente bancário digital.

Funcionalidades

    Autocadastro
    Login
  • Gerenciamento de contas bancárias
  • Registro de transações
  • Relatórios financeiros
